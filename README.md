This repository contains the data and the software to analyze the data. 

Folder Structure

Analysis: Contains the analysed data and figures.

calibration_latest: Contains a text file with the values of the gamma exponents 
used to create gamma corrected RGB values to be presented on the experimental monitor.


Data: Contains the data. The data is stored in data folders named after each observer.

Utilities: Contains the python scripts to analyze the data. See the README.md 
file in the folder for more information.
